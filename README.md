# 🎲 1. Password Generator

## 📌 How It Works

1. User enters the desired password length.
2. User selects whether to include:

   * Letters
   * Numbers
   * Symbols
3. The program generates a random password.
4. Password strength is analyzed and displayed.

## 🔧 Concepts Used

* Random number generation (`random`)
* String manipulation (`string`)
* Functions
* Conditional statements
* User input validation

## 🎯 What You Learn

* Python fundamentals
* Working with libraries
* Input validation
* Modular programming
* Security basics (strong passwords)

## 🚀 Real-World Use

Used in:

* Password managers
* Account creation systems
* Security tools

(PROJECT - @2)
# 🎮 2. Rock Paper Scissors Game

## 📌 How It Works

1. User selects Rock, Paper, or Scissors.
2. Computer randomly chooses a move.
3. Game compares both choices.
4. Winner is determined.
5. Scores are updated.
6. Match ends after Best of 3 or Best of 5.

## 🔧 Concepts Used

* GUI development (`CustomTkinter`)
* Event-driven programming
* Randomization
* Sound effects
* Animations (`after()`)
* Score management

## 🎯 What You Learn

* GUI development
* Game logic
* User interaction
* State management
* Multimedia integration

## 🚀 Real-World Use

Used in:

* Casual games
* Interactive applications
* Learning game development

# 📞 3. Contact Management System

## 📌 How It Works

## 🎯 What You Learn

* File management
* User interface design

## 🚀 Real-World Use
